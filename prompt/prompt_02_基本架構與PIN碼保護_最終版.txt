1. 建立 React 專案
   - 使用 Vite + React
   - 安裝套件：@supabase/supabase-js、react-router-dom
   - 建立 .env 檔，填入上面的 Supabase 連線資訊

2. 建立 Supabase 客戶端連線檔
   - 路徑：src/lib/supabase.js
   - 用環境變數初始化 Supabase 連線

3. 建立頁面路由結構（src/App.jsx）
   以下是所有頁面，先建立空白頁面佔位就好，內容之後再填：
   - /                首頁（行事曆 + 各模組入口）
   - /todos           老闆待辦清單（需要 PIN）
   - /notes           老闆專屬備注（需要 PIN）
   - /projects        專案進度看板（需要 PIN）
   - /checklist       展覽清點表（不需要 PIN）
   - /meetings        會議記錄（不需要 PIN）

4. 建立 PIN 碼保護機制
   規則：
   - PIN 碼是 4 位數，儲存在 .env 檔裡（VITE_PIN_CODE=你設定的PIN碼）
   - 使用者進入需要 PIN 的頁面時，顯示輸入框
   - 輸入正確 → 解鎖，把解鎖狀態存到 localStorage
   - localStorage 裡記錄解鎖的時間戳記
   - 每次進入需要 PIN 的頁面，檢查時間戳記是否在 24 小時內
   - 24 小時內 → 直接進入，不用重新輸入
   - 超過 24 小時 → 清除時間戳記，要求重新輸入 PIN
   - 輸入錯誤 → 顯示「PIN 碼錯誤，請再試一次」，不做其他限制

5. 建立 PIN 碼輸入元件（src/components/PinGate.jsx）
   - 這是一個包裝元件，套在需要保護的頁面外面
   - 未解鎖：顯示 PIN 碼輸入畫面
   - 已解鎖：直接顯示頁面內容
   - 輸入畫面：4格輸入框，輸入完自動送出，不需要按確認按鈕

6. 建立暫時首頁（src/pages/Home.jsx）
   - 先放一個簡單的標題「公司任務中心」
   - 列出六個功能的連結按鈕
   - 需要 PIN 的按鈕旁邊顯示一個鎖頭圖示
